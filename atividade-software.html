Projeto de Pesquisa: Categorias de Software

Unidade Curricular: Conceitos de Software

Professor: Elielson da Silva Mourão Curso: Instalador e Reparador de Redes

Prezados alunos, este projeto visa aprofundar os conhecimentos sobre as três grandes categorias de software. A pesquisa deve ser detalhada e cobrir os seguintes pontos:

1. Sistemas Operacionais (SO)

Conceitos Fundamentais

  • Definição: O que é e por que o computador não funciona sem ele?
  • Diferença entre sistemas de Código Aberto (ex: Linux) e Código Fechado (ex: Windows).

Plataformas

  • Diferenças entre Sistemas Operacionais Desktop e Mobile.
  • O conceito de Interface Gráfica (GUI) vs. Linha de Comando (CLI).

2. Softwares Aplicativos

Tipos e Licenciamento

  • Diferença entre Software Proprietário, Livre e Shareware.
  • O que é o modelo SaaS (Software como Serviço/Nuvem)?

Exemplos Práticos

  • Categorização: Editores de texto, navegadores, softwares de gestão e ferramentas técnicas.
  • A importância da interoperabilidade entre diferentes aplicativos.

3. Softwares Utilitários

Segurança e Otimização

  • Papel dos Antivírus e Firewalls de software.
  • Ferramentas de Backup e recuperação de dados.

Gestão de Sistema

  • Softwares de compactação de arquivos (WinRAR, 7-Zip).
  • Utilitários de diagnóstico e monitoramento de desempenho.

4. Requisitos de Entrega

  • Capa: Nome completo, curso, data e nome do professor.
  • Formato: Arquivo em PDF (organizado com títulos e imagens).
  • Atividade: Individual ou em Dupla.
