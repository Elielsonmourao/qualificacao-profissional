Projeto de Pesquisa: Categorias de Software

Explorando a inteligência por trás do hardware

1. Objetivo

O objetivo desta pesquisa é compreender as diferentes camadas de software que compõem o uso de um computador ou dispositivo móvel, diferenciando suas funções e importâncias no dia a dia.

2. Temas de Pesquisa

Sua pesquisa deve obrigatoriamente cobrir os três pilares abaixo:

A) Sistemas Operacionais (SO)

  • O que é um Sistema Operacional e qual sua principal função?
  • Diferença entre SO para computadores e SO para dispositivos móveis.
  • Exemplos: Windows, macOS, Linux, Android e iOS.

B) Softwares Aplicativos

  • Definição: Para que servem os aplicativos?
  • Categorias (Ex: Editores de texto, navegadores, jogos, planilhas).
  • A importância dos aplicativos na produtividade e entretenimento.

C) Softwares Utilitários

  • O que são utilitários e como eles auxiliam na manutenção do sistema?
  • Exemplos: Antivírus, compactadores de arquivos (WinRAR/7-Zip), desfragmentadores e softwares de backup.

3. O que deve ser entregue?

O projeto pode ser feito individualmente ou em dupla, contendo:

  1. Introdução: Uma breve explicação sobre o que é software.
  2. Desenvolvimento: O conteúdo pesquisado sobre os três temas acima, incluindo imagens e exemplos reais.
  3. Tabela Comparativa: Uma tabela listando 2 exemplos de cada categoria de software.
  4. Conclusão: Uma reflexão sobre como seria usar um computador sem um sistema operacional ou utilitários.

4. Critérios de Avaliação

  • Clareza: Explicação dos conceitos com suas próprias palavras.
  • Organização: Uso de tópicos, títulos e imagens ilustrativas.
  • Fontes: Listar os sites ou livros consultados ao final do trabalho.
